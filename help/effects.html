Effects

To check which effects you currently have on type /cpo. If you wish to only see a shorter list of effects you can set that list with the /cpl_prots comand. The prots need to be standardized by replacing spaces with underscores and removing any non-alphanumeric characters. Once you have your /cpl_prots set you can use the /cpl (shows a complete list of all prots, online and offline) and /cpm (shows only the missing prots) commands.

/cpl_prots lesser greater iron_will unpain
/cpl

If you wish to report your effects to the party you can append _p to the various effect checking commands. Party members can request your active prots by whispering 'prots' or 'proots' to you.
